How to Use

  • 1
    Set Options
    Choose between automatic IP selection or enter your favorite Cloudflare endpoint IP and port.
  • 2
    Click Generate
    Hit the generate button. It will create a unique Private Key and register it with Cloudflare.
  • 3
    Save Result
    Download the .conf file for PC or scan the QR code with your WireGuard mobile app.

How to Connect

WARP uses the WireGuard protocol. You can use this config in any WireGuard client.

  • Android & iOS
    Download 'WireGuard' from Store. Tap (+) and 'Scan from QR code'.
  • Windows & Mac
    Install 'WireGuard' and 'Add Tunnel' -> Import from file (.conf).
